# 必要経費管理アプリ（親子向け）

## 概要

本アプリは、**子供が立て替えた必要経費を親に請求・管理するためのWebアプリ**です。  
HTML / CSS / JavaScript のみで構成されており、サーバー不要でブラウザ上で動作します。

子供は支払った金額や用途、必要に応じてレシート画像を登録でき、  
親は支払いが完了した経費にチェックを入れることで管理できます。  
未払い経費の累計金額が自動で計算・表示されます。

---

## 想定利用シーン

- 子供が学校行事・部活動・教材購入などで立て替えた費用を管理したい
- 親が未払い金額を一目で把握したい
- 紙や口頭での請求を減らしたい

---

## 主な機能

### 1. ログイン機能
- ユーザー名・パスワードによるログイン
- ロール分離（子供 / 親）
- ブラウザ `localStorage` を利用した簡易認証

### 2. 経費登録（子供）
- 金額の入力
- 用途の入力
- レシート画像のアップロード（任意）
- 登録日を自動付与

### 3. 経費一覧表示
- 登録された経費を一覧表示
- 日付 / 金額 / 用途 / レシート有無 / 支払い状況を確認可能

### 4. 支払い管理（親）
- 支払い完了時にチェックボックスをON
- 子供側からは支払い状況を閲覧のみ可能

### 5. 累計金額表示
- **未払いの必要経費合計額**を常時表示
- 支払いチェックに応じて自動再計算

### 6. データ永続化
- `localStorage` に保存
- ブラウザをリロードしてもデータは保持される

## 要件定義

### 機能要件

| No | 機能 | 説明 |
|---|---|---|
| F-01 | ログイン | ユーザー認証を行う |
| F-02 | ロール管理 | 親・子供で権限を分ける |
| F-03 | 経費登録 | 子供が経費を登録できる |
| F-04 | 画像登録 | レシートを画像で保存できる |
| F-05 | 経費一覧 | 経費の一覧を表示する |
| F-06 | 支払い管理 | 親が支払い済みをチェックできる |
| F-07 | 合計表示 | 未払い経費の合計を計算・表示 |

---

### 非機能要件

- サーバー不要（フロントエンドのみ）
- 外部ライブラリ不使用
- PC / スマートフォンのブラウザで利用可能
- 学習用途・家庭内利用を想定（高セキュリティは対象外）

---

## 技術構成

## 制限事項・注意点

- パスワードは暗号化されていません（学習向け）
- ブラウザを変えるとデータは共有されません
- 複数家庭の同時管理は非対応
